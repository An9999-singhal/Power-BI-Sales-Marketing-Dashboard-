### 📊 Retail Sales & Customer Insights Dashboard (Power BI)

##🚀 Turning retail data into actionable insights using Power BI & DAX  

---

## 🧠 Project Overview  
This project presents an interactive Power BI dashboard built to analyze **retail sales performance, delivery efficiency, and customer behavior using the ShopNest dataset.  

The dataset consists of 7 integrated tables, which were cleaned, transformed, and modeled to ensure accurate analysis. The dashboard enables users to explore insights through interactive filters, drill-downs, and dynamic visualizations**.

---

## 🎯 Objective  
- Analyze -sales performance and trends 
- Evaluate -delivery efficiency and delays
- Understand -  customer behavior and payment preferences  
- Enable **data-driven decision-making


## 📊 Key Features  
- 📈 Sales trend analysis (time-based)  
- 🚚 Delivery performance tracking  
- 💳 Payment method insights  
- 🛍️ Product category performance  
- 🔍 Interactive filters & drill-downs  

---

## 💡 Key Insights  
- 📅 Peak sales observed in Q2
- 🏆 Top-performing product categories identified  
- 💳 74% of customers prefer credit card payments
- 📦 Delivery delays highlight operational inefficiencies

# Power-BI-Sales-Marketing-Dashboard-
First project on Github
This project presents an interactive Power BI dashboard built to analyze retail sales performance, delivery efficiency, and customer behavior using the ShopNest dataset. The objective was to derive actionable insights that could support data-driven decision-making in a retail business environment.  

The dataset consisted of 7 integrated tables, which were cleaned, transformed, and modeled to establish relationships for accurate analysis. Using Power BI and DAX, I developed dynamic visualizations to track key metrics such as sales trends, order delays, payment methods, and product category performance. Features like filters and drill-downs enable deeper exploration of the data and enhance user interactivity.  

Through this analysis, I identified key business insights, including peak sales in Q2, top-performing product categories, and a strong customer preference for credit card payments (74%). The dashboard helps in understanding customer purchasing patterns and operational bottlenecks, making it useful for improving sales strategies and delivery performance.  

This project demonstrates my ability to work with end-to-end data analytics processes, from data modeling to visualization, while translating business problems into meaningful insights using Power BI.
